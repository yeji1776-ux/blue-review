Blue Review 사용 가이드

블로거 협찬 관리의 모든 것, 한눈에 알아보세요.

1 시작하기

회원가입

이메일로 가입하면 닉네임을 함께 설정할 수 있습니다.

이메일 · 비밀번호 · 닉네임 입력

가입 화면에서 이메일, 비밀번호, 닉네임을 입력하고 가입하기를 누릅니다.

이메일 인증 (OTP)

입력한 이메일로 6자리 인증 코드가 발송됩니다. 코드를 입력해 인증을 완료하면 가입이 완료됩니다.

로그인 방법

이메일 로그인

가입한 이메일과 비밀번호로 로그인합니다.

Google 로그인

Google 계정으로 원클릭 로그인이 가능합니다.

생체인증 로그인 (Face ID / 지문)

한 번 로그인 후 프로필 기본설정에서 생체인증을 활성화하면, 다음부터 Face ID 또는 지문으로 빠르게 로그인할 수 있습니다.

자동 로그인

로그인 화면의 자동 로그인 체크박스를 선택하면 앱을 닫아도 로그인 상태가 유지됩니다.

게스트 모드

회원가입 없이도 앱의 주요 기능을 체험할 수 있습니다. 게스트 모드에서는 예시 스케줄이 표시되어 화면 구성을 미리 확인할 수 있습니다. 로그인 후에는 예시 데이터가 표시되지 않으며, 직접 등록한 스케줄만 보입니다.

TIP: 본격적으로 협찬을 관리하려면 회원가입을 권장합니다. 게스트 모드에서 작성한 데이터는 로그인 후 유지되지 않습니다.

2 홈 화면

인사말

홈 화면 상단에 닉네임이 포함된 인사말이 표시됩니다. 예: "안녕하세요! 하레님 :) 오늘 예정된 일정이 없어요" 또는 오늘 체험일이 있는 스케줄 수가 함께 안내됩니다.

촬영지수

현재 날씨와 위치를 기반으로 오늘의 촬영 적합도를 알려줍니다. 위치 접근 권한을 허용하면 현재 날씨를 기반으로 점수가 자동 계산됩니다. 맑은 날일수록 높은 점수가 표시됩니다.

TIP: 촬영지수는 홈 화면에서만 확인할 수 있습니다.

마감 알림 배너

신청 마감이 D-3 ~ D-day인 스케줄이 있으면 홈 화면 상단에 알림 배너가 표시됩니다. 배너를 탭하면 해당 스케줄 상세 화면으로 바로 이동합니다.

Quick Copy

협찬 신청 시 자주 사용하는 자기소개 문구(블로그 URL, 인스타 URL 등)를 한 번의 탭으로 복사합니다. 프로필 > 플랫폼 설정에서 활성화한 채널만 표시됩니다.

TIP: 채널 URL이 등록되지 않은 상태에서 복사 버튼을 누르면 "입력된 주소가 없습니다" 팝업이 나타납니다. 프로필에서 주소를 먼저 입력해 주세요.

신청문구 템플릿

협찬 신청 시 사용할 자기소개 문구를 여러 개 미리 등록해 두고 상황에 맞게 골라 사용할 수 있습니다. 템플릿은 추가, 수정, 삭제가 자유롭습니다.

오늘의 스케줄

오늘 체험 예정인 스케줄이 홈 화면 하단에 카드 형태로 표시됩니다. 카드를 탭하면 해당 스케줄 상세 화면으로 이동합니다.

3 스케줄 등록 (Smart Parser)

협찬 사이트의 캠페인 내용을 복사해서 앱에 붙여넣거나, 업체가 보낸 PDF · Word 파일을 업로드하면 AI가 브랜드명·마감일·지역·제공 내용 등을 자동으로 추출합니다. 두 가지 방법 모두 아래에서 예시로 확인하세요!

📋 방법 1 — 텍스트 복사·붙여넣기

협찬 사이트(레뷰/미블/리뷰노트 등)의 캠페인 페이지를 복사해서 붙여넣기

📄 방법 2 — PDF · Word 파일 업로드

업체가 이메일·카톡으로 보낸 원본 파일(.pdf / .docx)을 그대로 업로드 (최대 20MB)

플랫폼 선택

이 협찬을 어떤 채널로 진행할지 선택합니다. 블로그 블로그+클립 인스타 릴스 페이스북 유튜브 복수 선택 가능합니다.

내용 확인 후 저장

자동 추출된 브랜드·카테고리·마감일 등을 확인하고, 필요한 경우 수정한 뒤 저장하기를 누릅니다.

TIP: 분석 결과가 비어 있거나 부정확하면 직접 수정하면 됩니다. 복사한 내용에 브랜드명, 날짜, 지역 등이 명확하게 포함되어 있을수록 더 정확하게 분석됩니다.

📄 파일 업로드가 더 편한 경우

  • • 업체가 공식 캠페인 안내서(.pdf)를 보낸 경우
  • • 신청 이후 전달받은 Word 상세 가이드(.docx)가 있는 경우
  • • 복사·붙여넣기 과정에서 서식이 깨지는 경우

업로드 즉시 자동 분석이 시작되고, 따로 "분석하기" 버튼을 누르지 않아도 결과 화면으로 이동합니다.

실제 화면 예시 — 배지 줄 · 체험일 · 지역 · 체험기간 · 리뷰마감이 한눈에 보입니다.

플랫폼 배지

브랜드(예: 리뷰노트) · 카테고리(예: 맛집) · 채널(블로그/인스타/릴스 등)이 차례로 표시됩니다.

체험일 설정 · 변경

실제 방문/체험 예정일을 설정합니다. 변경을 누르면 날짜·시간을 수정할 수 있습니다.

체험 메모

체험 중 떠오른 포인트, 메뉴 이름, 분위기 키워드를 그 자리에서 기록해두면 리뷰 작성 시 빠르게 활용할 수 있습니다.

공정위 문구 · 캘린더 추가

공정위 버튼은 저장된 협찬표시 URL을 복사합니다. 캘린더 추가/수정은 뒤의 "Google Calendar 연동" 섹션 참고.

일정 공유

상세 카드를 이미지(PNG)로 저장합니다. 가족·클라이언트에게 일정 공유용으로 내보낼 때 유용합니다.

완료 처리

리뷰 작성이 끝나면 완료 상태로 변경합니다. 완료된 스케줄은 스케줄 관리 탭의 완료 목록에서 확인할 수 있습니다.

📄 신청문구 복사

상세 화면 상단의 신청문구 버튼을 누르면 저장해 둔 신청문구 목록이 팝업으로 표시됩니다. 원하는 문구를 탭하면 즉시 클립보드에 복사됩니다.

🗂 2단계 마감일 (초안 마감 + 최종 마감)

기자단 · 제품 카테고리처럼 초안을 먼저 보낸 뒤 수정 반영해서 최종 업로드하는 캠페인은, 초안 마감과 최종 마감이 각각 따로 관리됩니다.

① 초안 마감

파서가 "초안 제출 마감" 키워드를 잡으면 자동으로 채워집니다. 아직 지나지 않으면 D-day 배지가 초안 기준으로 표시됩니다.

② 최종 마감

초안 마감일이 지나면 자동으로 D-day 기준이 최종 마감으로 전환됩니다. 수정 모드에서 직접 입력·수정도 가능합니다.

예시 — 기자단: 초안 3/20 → 최종 3/25. 오늘이 3/18이면 카드에 "D-2 (초안)"이 표시되고, 3/22부터는 "D-3 (최종)"으로 바뀝니다.

🗓 Google Calendar 연동

프로필에서 구글 계정을 연결한 상태라면, 상세 카드에 캘린더 추가 / 캘린더 수정 버튼이 나타납니다. 한 번 누르면 체험일 · 초안 마감 · 최종 마감이 구글 캘린더에 각각 등록됩니다.

캘린더 추가

처음 연동할 때 표시되는 버튼. 체험일은 지정 시각 이벤트로, 마감일은 종일 이벤트로 등록됩니다.

캘린더 수정

이미 등록된 스케줄의 날짜를 바꾸면 이 버튼으로 구글 캘린더 이벤트도 같이 업데이트됩니다.

스케줄 삭제 시

앱에서 스케줄을 삭제하면 연결된 구글 캘린더 이벤트도 함께 삭제됩니다.

TIP: 구글 로그인 토큰이 만료되면 버튼이 비활성화됩니다. 프로필에서 다시 연결해 주세요.

✏ 수정 모드 · 파일로 한 번에 업데이트

상세 화면 우측 상단 연필 아이콘을 누르면 수정 모드로 전환됩니다. 브랜드 · 제목 · 주소 · 마감일 등 모든 필드를 직접 편집할 수 있고, PDF · Word 파일로 한 번에 덮어쓰기도 가능합니다.

PDF · Word 파일로 업데이트

업체에서 캠페인 안내서가 바뀌었을 때 원본 파일만 올리면 필드가 다시 채워집니다

TIP: 파일 업데이트는 기존 값을 덮어쓰기합니다. 체험일·신청문구 등 앱에서 따로 관리하는 값은 유지되고, 캠페인 안내서에 포함된 브랜드·주소·마감·키워드 등의 필드만 새로 반영됩니다.

스케줄 삭제

상세 화면 상단의 🗑 삭제 버튼을 누르면 확인 팝업이 표시됩니다. 확인 후 삭제가 진행되며, 삭제된 스케줄은 복구할 수 없습니다. 구글 캘린더에 연동된 이벤트도 함께 삭제됩니다.

5 캘린더

캘린더 탭에서 모든 스케줄을 달력 형태로 한눈에 확인할 수 있습니다.

날짜 표시 점(Dot) 안내

빨간 점 — 신청 마감일

파란 점 — 체험 예정일

회색 점 — 리뷰 완료

날짜 클릭하여 상세 확인

캘린더에서 날짜를 탭하면 해당 날짜의 스케줄 목록이 아래에 표시됩니다. 목록에서 항목을 탭하면 상세 화면으로 이동합니다.

6 스케줄 관리 & 분석

연도 / 월 / 플랫폼 필터

연도와 월을 선택해 원하는 기간의 스케줄만 조회할 수 있습니다. 플랫폼 필터를 사용하면 블로그, 인스타, 릴스 등 특정 채널의 협찬만 골라볼 수 있습니다.

TIP: 플랫폼 필터에는 프로필에서 활성화한 채널만 표시됩니다. 원하는 채널이 보이지 않으면 프로필 > 플랫폼 설정에서 해당 채널을 켜주세요.

진행중 / 완료 목록

탭을 전환하여 현재 진행중인 스케줄과 완료된 스케줄을 분리해서 확인할 수 있습니다.

월간 분석

선택한 월의 협찬 현황을 분석합니다.

  • • 플랫폼별 분석 — 블로그, 인스타, 릴스 등 채널별 협찬 건수 확인
  • • 브랜드별 분석 — 어떤 브랜드와 가장 많이 협업했는지 확인
  • • 카테고리별 분석 — 맛집, 뷰티, 숙박 등 카테고리별 비율 확인

연간 분석

1년 단위의 종합 분석 데이터를 제공합니다.

  • • 트렌드 — 월별 협찬 건수 추이를 그래프로 확인
  • • 완료율 — 전체 스케줄 대비 리뷰 완료 비율
  • • 플랫폼 & 브랜드 비율 — 연간 채널·브랜드별 협업 비중 확인

7 도구

글자 수 측정

블로그 리뷰 작성 시 글자 수 조건을 충족하는지 빠르게 확인합니다. 텍스트를 입력하거나 붙여넣으면 공백 포함 / 제외 글자 수가 실시간으로 표시됩니다.

글 저장하기

작성한 글을 작성글 탭에 저장합니다. 저장 시 "작성글에 저장되었습니다" 팝업이 표시됩니다.

복사

입력한 전체 텍스트를 클립보드에 복사합니다.

닫기

글자 수 측정 화면을 닫습니다. 다시 열면 내용이 초기화됩니다.

TIP: 글자 수 측정 화면은 열 때마다 새로 시작됩니다. 작성한 글을 나중에 다시 보려면 글 저장하기를 눌러 작성글에 보관하세요.

작성글

저장된 리뷰 글을 모아두는 공간입니다. 글의 첫 번째 줄이 제목으로 표시됩니다.

글 목록 확인

글자 수 측정에서 저장한 글이 작성글 탭에 목록으로 표시됩니다. 첫 줄이 제목으로 보입니다.

수정

목록에서 글을 탭하면 수정 화면이 열립니다. 수정 후 저장하면 새 글로 추가되는 것이 아니라 해당 글이 그대로 업데이트됩니다.

해시태그 모음

자주 사용하는 해시태그를 카테고리별로 정리하여 관리합니다.

카테고리 추가 / 삭제

맛집, 뷰티, 여행 등 원하는 카테고리를 자유롭게 추가하거나 삭제합니다.

태그 편집 및 복사

각 카테고리에서 해시태그를 추가, 수정, 삭제합니다. 태그를 탭하면 클립보드에 복사됩니다.

TIP: 카테고리별로 태그를 정리해 두면 리뷰 작성 시 필요한 태그를 빠르게 찾아 복사할 수 있습니다.

8 내 프로필

프로필 탭은 플랫폼 설정과 기본 설정 두 개의 탭으로 구성되어 있습니다.

플랫폼 설정 탭

사용 중인 SNS 채널을 켜고 끄며, 각 채널의 URL을 등록합니다.

채널 활성화 토글

블로그, 블로그+클립, 인스타, 릴스, 페이스북, 유튜브 중 사용하는 채널만 켭니다. 비활성화된 채널은 Quick Copy와 스케줄 관리 플랫폼 필터에 표시되지 않습니다.

URL 입력

활성화된 각 채널의 URL(블로그 주소, 인스타그램 링크 등)을 입력합니다. 블로그+클립은 URL 입력이 필요하지 않습니다.

Quick Copy 항목 확인 & 저장

홈 화면 Quick Copy에 표시할 문구를 확인하고 저장합니다. URL이 등록되지 않은 채널의 복사 버튼을 누르면 "입력된 주소가 없습니다" 팝업이 표시됩니다.

기본 설정 탭

계정 관련 설정을 관리합니다.

계정 이메일 확인

현재 로그인된 계정의 이메일을 확인합니다.

생체인증 로그인

Face ID 또는 지문 인증을 활성화 / 비활성화합니다. 활성화하면 다음 로그인부터 생체인증으로 빠르게 접속할 수 있습니다.

비밀번호 변경

현재 비밀번호를 새 비밀번호로 변경합니다.

계정 탈퇴

계정을 영구 삭제합니다. 탈퇴 후에는 모든 데이터가 삭제되며 복구할 수 없습니다.
