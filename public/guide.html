B

Blue Review 사용 가이드

블로거 협찬 관리의 모든 것, 한눈에 알아보세요.

1 시작하기

회원가입 / 로그인

Blue Review는 세 가지 방법으로 간편하게 시작할 수 있습니다.

이메일 가입

이메일 주소와 비밀번호를 입력하여 회원가입 후 로그인합니다.

Google 로그인

Google 계정으로 원클릭 로그인이 가능합니다.

카카오톡 로그인

카카오톡 계정을 연동하여 빠르게 시작하세요.

게스트 모드

회원가입 없이도 앱을 둘러볼 수 있습니다. 게스트 모드에서는 기본 기능을 체험할 수 있으며, 데이터 저장이나 동기화 기능은 제한됩니다. 본격적으로 사용하시려면 회원가입을 권장합니다.

TIP: 게스트 모드에서 작성한 데이터는 회원가입 후에도 유지되지 않을 수 있으니, 중요한 정보는 로그인 후 입력해 주세요.

2 홈 화면

촬영지수

현재 날씨와 위치를 기반으로 오늘의 촬영 적합도를 알려줍니다. 사용을 위해 날씨 정보와 위치 접근 권한에 동의해 주세요.

TIP: 위치 권한을 허용하면 현재 위치의 날씨를 기반으로 촬영지수가 자동 계산됩니다. 맑은 날일수록 높은 점수가 표시됩니다.

마감 알림 배너

신청 마감이 D-3 ~ D-day인 스케줄이 있으면 홈 화면 상단에 알림 배너가 표시됩니다. 배너를 탭하면 해당 스케줄로 바로 이동합니다.

Quick Copy 사용법

협찬 신청 시 자주 사용하는 자기소개 문구를 한 번의 탭으로 복사할 수 있습니다.

홈 화면에서 Quick Copy 영역을 확인합니다.

복사하고 싶은 문구를 탭하면 자동으로 클립보드에 복사됩니다.

협찬 사이트의 신청 페이지에 붙여넣기 하세요.

신청문구 템플릿 관리

여러 개의 신청문구 템플릿을 미리 등록해 두고, 상황에 맞게 골라 사용할 수 있습니다. 템플릿은 추가, 수정, 삭제가 자유롭습니다.

브랜드별 스케줄 카드

등록된 협찬 스케줄이 브랜드별로 카드 형태로 표시됩니다. 각 카드에는 브랜드명, 카테고리, 마감일 등 핵심 정보가 한눈에 보입니다.

3 스케줄 등록 (Smart Parser)

협찬 사이트의 내용을 복사해서 붙여넣기만 하면 AI가 자동으로 분석하여 스케줄을 등록합니다.

하단 + 버튼 누르기

화면 하단의 + 버튼을 눌러 스케줄 등록 화면으로 이동합니다.

협찬 내용 복사 → 붙여넣기

협찬 사이트(레뷰, 미블, 리뷰노트 등)에서 캠페인 내용을 전체 복사한 뒤, 입력창에 붙여넣기 합니다.

AI 자동 분석

붙여넣기 하면 AI(Smart Parser)가 텍스트를 분석하여 캠페인명, 마감일, 지역, 제공 내용 등을 자동으로 추출합니다.

브랜드 / 카테고리 선택

자동 추출된 정보를 확인하고 해당 캠페인의 브랜드와 카테고리를 선택합니다.

필드 수정 후 저장

AI가 추출한 정보가 정확한지 확인하고, 필요시 수정한 뒤 저장 버튼을 누르면 등록 완료입니다.

TIP: 복사할 내용이 많을수록 AI 분석 정확도가 높아집니다. 캠페인 페이지 전체를 복사하는 것을 추천합니다.

4 스케줄 상세 보기

홈 화면의 스케줄 카드를 클릭하면 상세 정보를 확인하고 관리할 수 있습니다.

카드 클릭하여 상세 보기

스케줄 카드를 탭하면 캠페인의 전체 정보가 표시됩니다.

체험일 설정

실제 체험 예정일을 설정하여 일정을 관리합니다.

체험 느낌 메모

체험 후 느낌이나 리뷰에 쓸 핵심 키워드를 메모합니다.

공정위 문구 URL

공정거래위원회 문구가 포함된 URL을 저장합니다.

블로그 리뷰 글 링크

작성 완료된 블로그 리뷰 글의 URL을 등록합니다.

리뷰 등록 완료 처리

리뷰 작성이 끝나면 완료 상태로 변경합니다.

이미지 저장 (다운로드)

스케줄 관련 이미지를 기기에 저장할 수 있습니다.

TIP: 체험 느낌 메모를 미리 작성해 두면 리뷰 글 작성 시 큰 도움이 됩니다.

5 캘린더

캘린더 탭에서 모든 스케줄을 달력 형태로 한눈에 확인할 수 있습니다.

날짜 표시 점(Dot) 안내

빨간 점 — 신청 마감일

파란 점 — 체험 예정일

회색 점 — 리뷰 완료

날짜 클릭하여 상세 확인

캘린더에서 날짜를 탭하면 해당 날짜의 스케줄 목록이 아래에 표시됩니다. 목록에서 항목을 탭하면 상세 페이지로 이동합니다.

6 전체 관리 & 분석

연도/월별 필터링

상단의 연도 및 월 선택기를 사용하여 원하는 기간의 스케줄만 필터링하여 볼 수 있습니다.

진행중 / 완료 목록

탭을 전환하여 현재 진행중인 스케줄과 완료된 스케줄을 분리하여 확인할 수 있습니다.

월간 분석

선택한 월의 협찬 현황을 분석합니다.

  • • 브랜드별 분석 — 어떤 브랜드와 가장 많이 협업했는지 확인
  • • 카테고리별 분석 — 맛집, 뷰티, 숙박 등 카테고리별 비율 확인

연간 분석

1년 단위의 종합 분석 데이터를 제공합니다.

  • • 트렌드 — 월별 협찬 건수 추이를 그래프로 확인
  • • 완료율 — 전체 스케줄 대비 리뷰 완료 비율
  • • 브랜드 비율 — 연간 브랜드별 협업 비중 확인

7 도구

글자 수 측정기

블로그 리뷰 작성 시 글자 수 조건을 충족하는지 빠르게 확인할 수 있습니다. 텍스트를 입력하거나 붙여넣기 하면 공백 포함/제외 글자 수가 실시간으로 표시됩니다.

해시태그 모음

자주 사용하는 해시태그를 카테고리별로 정리하여 관리합니다.

카테고리 추가 / 삭제

맛집, 뷰티, 여행 등 원하는 카테고리를 자유롭게 추가하거나 삭제할 수 있습니다.

태그 편집

각 카테고리 안에서 해시태그를 추가, 수정, 삭제합니다. 태그를 탭하면 클립보드에 복사됩니다.

TIP: 카테고리별로 해시태그를 정리해 두면 리뷰 작성 시 필요한 태그를 빠르게 찾아 복사할 수 있습니다.

8 내 프로필

SNS 주소 등록

네이버 블로그, 인스타그램 등 자신의 SNS 주소를 등록해 두세요. 등록된 주소는 Quick Copy에서 자동으로 연동되어 신청문구에 포함할 수 있습니다.

프로필 탭으로 이동합니다.

SNS 주소 항목에 블로그 URL, 인스타그램 ID 등을 입력합니다.

저장하면 Quick Copy 신청문구에 자동 반영됩니다.

Quick Copy 연동

프로필에 등록한 SNS 주소와 블로그 정보가 Quick Copy 신청문구 템플릿에 자동으로 연동됩니다. 프로필 정보를 업데이트하면 모든 템플릿에 자동 반영되므로, 항상 최신 정보를 유지해 주세요.

TIP: 프로필을 완성해 두면 협찬 신청 시 매번 SNS 주소를 입력하는 번거로움을 줄일 수 있습니다.
